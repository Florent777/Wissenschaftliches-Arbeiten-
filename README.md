# Wissenschaftliches Arbeiten – Chancen und Risiken von ETFs für Privatanleger

Gemeinsames Repository von Florent und Jerome für die Wissenschaftliche Arbeit (FHDW) inklusive Abschlusspräsentation.

## Schnellstart

1. `git pull`
2. [`Status/Fortschritt.md`](./Status/Fortschritt.md) lesen – aktueller Stand & offene Punkte
3. [`Gliederung.md`](./Gliederung.md) lesen – Struktur & wer schreibt was
4. Inhalte in [`Entwurf/`](./Entwurf/) als Markdown erarbeiten (mit Claude Code o.ä.), fertigen Text danach in [`Word/`](./Word/) übertragen
5. Vor Schluss: Änderung in `Status/Fortschritt.md` eintragen, committen, pushen

Alle Regeln für die Zusammenarbeit – auch mit KI-Assistenten – stehen in [`CLAUDE.md`](./CLAUDE.md).

## Struktur

| Pfad | Zweck |
|---|---|
| [`Gliederung.md`](./Gliederung.md) | Struktur/Inhaltsverzeichnis, Verantwortlichkeiten |
| [`Status/Fortschritt.md`](./Status/Fortschritt.md) | Koordinations-Log |
| [`Quellen/Literaturverzeichnis.md`](./Quellen/Literaturverzeichnis.md) | gemeinsame Quellenliste |
| [`Entwurf/`](./Entwurf/) | Markdown-Entwürfe pro Kapitel |
| [`Word/`](./Word/) | finale .docx-Dateien (Abgabeformat) |
| [`Praesentation/`](./Praesentation/) | Vorbereitung der Abschlusspräsentation |

## Noch zu erledigen

- [ ] Sichtbarkeit des Repos prüfen (privat empfohlen, solange die Arbeit unfertig ist)
